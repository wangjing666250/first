# XX-UI (Pillar-UI)

A Vue 3 + TypeScript component library monorepo for admin products.

## Quick Start

```bash
pnpm install
pnpm dev
```

## Workspace

- `packages/pillar-ui`: component library package
- `packages/theme`: design token and global CSS variable package
- `play`: local playground app
- `docs`: interview drill notes
- `scripts`: affected build and release checklist

## Main Commands

- `pnpm dev`: run playground
- `pnpm build`: build all packages
- `pnpm build:affected`: build changed packages (with fallback)
- `pnpm test`: run component tests
- `pnpm test:coverage`: run coverage
- `pnpm lint`: lint workspace
